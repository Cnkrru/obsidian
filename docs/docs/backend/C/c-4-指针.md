# C语言指针

## 1. 指针基础

### 指针定义
```c
类型 *指针名;
```
**说明**：声明指针变量

### 取地址
```c
&变量名
```
**说明**：获取变量的内存地址

### 解引用
```c
*指针名
```
**说明**：通过指针访问变量值

### 空指针
```c
int *p = NULL;
```
**说明**：不指向任何有效内存的指针

### 野指针
```c
int *p; // 野指针
```
**说明**：未初始化的指针，指向无效内存

## 2. 指针进阶

### 指针运算
```c
指针 ± 整数
```
**说明**：指针的加减操作

### 指针数组
```c
类型 *数组名[大小];
```
**说明**：存储指针的数组

### 数组指针
```c
类型 (*指针名)[大小];
```
**说明**：指向数组的指针

### 多级指针
```c
类型 **指针名;
```
**说明**：指向指针的指针
