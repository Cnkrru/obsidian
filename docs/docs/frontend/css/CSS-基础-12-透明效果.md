## 1. 透明效果实现

### 1.1 方法一：opacity属性

```css
.element {
  opacity: 0.5; /* 半透明效果 */
}
```

### 1.2 方法二：rgba颜色

```css
.element {
  /* 半透明背景色 */
  background-color: rgba(52, 152, 219, 0.5);
  /* 半透明文本颜色 */
  color: rgba(0, 0, 0, 0.7);
}
```
