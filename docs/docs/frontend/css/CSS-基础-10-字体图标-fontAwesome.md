## 1. 字体图标

### 1.1 实现方法

#### 方法一：使用在线图标库（如Iconfont）

1. 注册并登录Iconfont
2. 创建项目并添加图标
3. 下载并引入字体文件
4. 使用图标
  ```html
  <i class="iconfont icon-home"></i>
  ```

#### 方法二：自定义字体图标

1. 设计SVG格式的图标
2. 使用工具将SVG转换为字体文件
3. 引入字体文件并定义CSS类
4. 使用图标
  ```html
  <i class="icon icon-home"></i>
  ```
