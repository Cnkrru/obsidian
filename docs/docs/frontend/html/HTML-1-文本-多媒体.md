## HTML基础标签与操作
---
## 1. 文本相关标签

- **h**：标题（独占一行，加粗）

- **p**：段落（独占一行）

---
## 2. 多媒体相关标签

### 图像标签

- **img**（单标签）
  - **src**：图像路径（先写./，vscode有提示）
  - **alt**：失败替换文本
  - **title**：图像名称
  - **width**：图像宽度（一般不用，用CSS来确定）
  - **height**：图像高度（同上）

### 音频标签

- **audio**
  - **src**：路径
  - **controls**：控制面板（不需要=填写属性值）
  - **loop**：循环播放
  - **autoplay**：自动播放

### 视频标签

- **video**
  - **src**：路径
  - **controls**：控制面板
  - **loop**：循环
  - **muted**：静音播放
  - **autoplay**：自动播放（自动需要配合静音播放一起使用）

### 文本格式化标签

- **strong/b**：加粗

- **em/i**：倾斜

- **ins/u**：下划线

- **del/s**：删除线

### 单标签

- **br**：换行

- **hr**：水平线
