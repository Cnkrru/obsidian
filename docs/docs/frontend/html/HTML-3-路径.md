#HTML
## 链接与路径

### 超链接标签

- **a**
  - **href**：URL/本地相对路径（初期不知道跳转到哪里，href=#，空连接）
  - **target**：_打开方式

### 路径

- **相对路径**
  - 从当前位置出发，找本地
  - 表示方法：
    - 文件夹名/：进入该文件夹
    - ./：进入当前文件夹
    - ../：进入上一级文件夹
    - 每多一个点，多进入一级

- **绝对路径**
  - 从盘符位置出发，找网页
  - 本地文件不用

- **路径分隔符**
  - 所有系统默认：/
  - Windows默认：\，但全使用/
