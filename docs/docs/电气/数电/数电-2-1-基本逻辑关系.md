## 真值表(Truth Table)

| A | B | Y |

| 0 | 0 | 0 |
| 0 | 1 | 0 |
| 1 | 0 | 0 |

====================================
## 与运算(AND Operation)
- 所有输入均为1=>输出为1(输入不限制)
- 符号：&
- L=AB
- 交集运算(串联)

## 或运算(OR Operation)
- 任意输入为1=>输出为1(输入不限制)
- 符号：|
- L=A+B
- 并集运算(并联)

## 非运算(NOT Operation)
- 输入为1=>输出为0(输入只有1个)
- 符号：!
- L=!A
- 补集运算(开关)

====================================
## 与非/或非/与或非(AND-NOT/OR-NOT/AND-OR-NOT)
- 与非:AB取反
- 或非:A+B取反
- 与或非:AB+CD取反

=====================================
## 异或运算(XOR Operation)（常见）
- 两元素取值相同=>输出为0（一般只用2输入）
- 两元素取值不同=>输出为1
- L=A!B+!AB

## 同或运算(SOR Operation)(暂时没什么用)
- 两元素取值相同=>输出为1
- 两元素取值不同=>输出为0
- L=A+B+!A!B
