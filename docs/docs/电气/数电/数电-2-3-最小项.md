## 什么是最小项？

n个变量X₁、X₂、……、Xₙ的最小项是n个因子的乘积，每个变量都以它的原变量或反变量的形式在乘积项中出现，且仅出现一次。

**以A、B、C作为3个逻辑输入变量为例**：
- ¯A·¯B·¯C ✔️
- ¯A·¯B·C ✔️
- A·¯B ❌ （缺少变量C）
- A·¯B·C·¯A ❌ （变量A出现两次）
- A·(B+C) ❌ （不是乘积项）

## 最小项的表示方法

**原变量——逻辑"1"；反变量——逻辑"0"**

| 最小项的变量表示形式 | 变量取值 | 最小项的编号表示形式 |
| :- | :- | :- |
| ¯A·¯B·¯C | A=0, B=0, C=0 | m₀ |
| ¯A·¯B·C | A=0, B=0, C=1 | m₁ |
| ¯A·B·¯C | A=0, B=1, C=0 | m₂ |
| ¯A·B·C | A=0, B=1, C=1 | m₃ |
| A·¯B·¯C | A=1, B=0, C=0 | m₄ |
| A·¯B·C | A=1, B=0, C=1 | m₅ |
| A·B·¯C | A=1, B=1, C=0 | m₆ |
| A·B·C | A=1, B=1, C=1 | m₇ |

## 最小项的性质

1. **性质1**：对于任意一个最小项，只有一组变量取值使得它的值为1，而在变量取其它各组值时，这个最小项的值都是0。

2. **性质2**：不同的最小项，使它的值为1的那一组变量取值也不同。

3. **性质3**：对于变量的任一组取值，任意两个最小项的乘积为0。

4. **性质4**：对于变量的任一组取值，全体最小项之和为1。

5. **性质5**：具有相邻性的两个最小项之和可以合并成一项并消去一对因子。

6. **性质6**：某一最小项若不包含在L中，则必在¬L中。

7. **性质7**：n个变量的逻辑函数有2ⁿ个最小项。

## 逻辑函数的标准与或式

### 标准与或式与真值表之间的关系

**真值表**

| A | B | C | Y | mᵢ |
| :- | :- | :- | :- | :- |
| 0 | 0 | 0 | 0 | m₀ |
| 0 | 0 | 1 | 1 | m₁ |
| 0 | 1 | 0 | 1 | m₂ |
| 0 | 1 | 1 | 1 | m₃ |
| 1 | 0 | 0 | 0 | m₄ |
| 1 | 0 | 1 | 1 | m₅ |
| 1 | 1 | 0 | 0 | m₆ |
| 1 | 1 | 1 | 0 | m₇ |

**标准与或式**
- Y = m₁ + m₂ + m₃ + m₅
- = Σm(1,2,3,5)
- = Σmᵢ (i=1,2,3,5)
- = ¯A·¯B·C + ¯A·B·¯C + ¯A·B·C + A·¯B·C

**说明**：若将真值表中函数值为0的最小项相加，便可得到其反函数¬Y的最小项之和形式。

### 什么是标准与或式？

也称**最小项之和的形式**。任何一个逻辑函数都可以表示成唯一的一组最小项之和形式，就称其为标准与或式。

**【重要提示】**：
一个逻辑功能的逻辑函数表达式形式是可以多种多样的，但是其标准与或式（或称最小项之和的形式）是唯一的！

### 标准与或式的表示方法

以3变量逻辑函数为例：输入变量分别用A、B、C表示。

**示例**：
Y = A·B·C + (A·B + B·C + A·B)
= A·(B + C) + A·B·B·C·A·B
= A·B + A·C + A·B·(B + C)·(A + B)
= A·B + A·C + A·B·(A·B + A·C + B·C)
= A·B + A·C + A·B + A·B·C
= A·B + A·C + A·B·C （普通与或式）

转换为标准与或式：
Y = A·B + A·C + A·B·C
= A·B·(C + ¯C) + A·(B + ¯B)·C + A·B·C
= A·B·C + A·B·¯C + A·B·C + A·¯B·C + A·B·C
= A·B·C + A·B·¯C + A·¯B·C
= m₇ + m₆ + m₅
= Σmᵢ (i=4,5,6)
= Σm(4,5,6) = Σ(4,5,6)
