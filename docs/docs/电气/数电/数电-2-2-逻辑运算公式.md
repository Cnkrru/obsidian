## 逻辑代数中的常用公式

1.  A + A·B = A 
   - 推导： A·(1 + B) = A · 1 = A 

2.  A + ¯A·B = A + B 
   - 推导： A + ¯A·B = (A + ¯A)·(A + B) = 1 · (A + B) = A + B 

3.  A·B + A·¯B = A 
   - 推导： A·(B + ¯B) = A · 1 = A 

4.  A·(A + B) = A 
   - 推导： A · A + A · B = A + A·B = A 

5.  A·B + ¯A·C + B·C = A·B + ¯A·C （多余项定理）
   - 推广： A·B + ¯A·C + B·C·D = A·B + ¯A·C 

6.  A · ¯(A·B) = A·¯B 
   - 推导： A · (¯A + ¯B) = A·¯A + A·¯B = A·¯B 
   - 应用摩根定理证明： ¯(A · A·B) = ¯A 

## 逻辑代数中的基本定理（或称规则）

### 1. 代入定理（或规则）
- **内容**：在任何一个包含变量A的逻辑等式中，若以另外一个逻辑式代入式中所有A的位置，则等式仍然成立。
- **适用范围**：利用代入规则可以将逻辑代数基本及常用公式中的变量用任意函数代替，从而推导出更多的等式。这些等式可直接作为公式使用，无须另加证明。
- **简言之**：拓展公式和定理时常用！

**示例**：试将二变量的摩根定理拓展为三变量的形式。
- 二变量摩根定理：
  -  ¯(A+B) = ¯A·¯B 
  -  ¯(A·B) = ¯A+¯B 
- 拓展为三变量：
  -  ¯(A+B+C) = ¯A·¯B·¯C 
  -  ¯(A·B·C) = ¯A+¯B+¯C 

### 2. 对偶定理（或规则）
- **原则**：若两个逻辑式相等，则它们的对偶式也相等；反之亦然。
- **适用范围**：证明两个逻辑式相等，可以通过证明对偶式相等来证。

**对偶式转换规则**：
-  1 ↔ 0 
-  0 ↔ 1 
-  + ↔ · 
-  · ↔ + 
- 逻辑变量不变
- 保证运算顺序不变→加括号
- 两变量及以上变量头上的大非号不动

**示例1**：证明公式  A+B·C = (A+B)·(A+C) 
- 对偶式：
  -  L₁' = A·(B+C) = A·B + A·C 
  -  L₂' = A·B + A·C 
- 因为  L₁' = L₂' ，所以  L₁ = L₂ 

**示例2**：已知原式  Y = ¯(¯A·B·C) + ¯C·¯D·(A·C + B·D) ，求其对偶式  Y' 
- 对偶式： Y' = (¯A+¯B+C)·(¯C+D)+(A+C)·(B+D) 

### 3. 反演定理（或规则）

**反函数转换规则**：
-  1 ↔ 0 
-  0 ↔ 1 
-  + ↔ · 
-  · ↔ + 
- 逻辑变量取反
- 保证运算顺序不变
- 两变量及以上变量头上的大非号不动

**示例**：已知原式  Y = ¯(¯A·B·C) + ¯C·¯D·(A·C + B·D) ，求其反函数  ¯Y 
- 反函数： ¯Y = (¯A+¯B+C)·(C+D)·(¯A+¯C)·(¯B+¯D)
