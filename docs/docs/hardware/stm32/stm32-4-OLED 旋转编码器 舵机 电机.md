# STM32

---
## OLED、旋转编码器、舵机、电机

## OLED配置

### 基本信息
- **供电**：3.3V
- **通信协议**：I2C
- **引脚定义**：
  - GND：接地
  - 3.3V：电源
  - SCL：I2C时钟同步线
  - SDA：I2C数据同步线

### 配置步骤
1. 开启I2C时钟
2. 配置GPIO为开漏输出
3. 初始化I2C
4. 发送初始化命令
5. 实现显示函数

## 旋转编码器

### 工作原理
- **用途**：测量旋转角度、速度和方向
- **角度测量**：根据A/B两相的边沿变化次数
- **速度测量**：根据边沿变化频率
- **方向判断**：根据A/B两相的相位差

### 配置步骤
1. 配置GPIO为输入模式
2. 配置外部中断
3. 在中断处理函数中判断旋转方向
4. 实现计数和速度测量

## 舵机

### 工作原理
- **控制方式**：通过PWM占空比控制输出角度
- **特性**：
  - 响应时间：0.12s/60°
  - 角度定位：每个角度都是固定位置
- **PWM参数**：
  - 周期：20ms
  - 高电平：0.5~2.5ms

### 角度对照表

| 脉冲宽度 | 输出角度 |
|---------|---------|
| 0.5ms   | -90°    |
| 1ms     | -45°    |
| 1.5ms   | 0°      |
| 2ms     | 45°     |
| 2.5ms   | 90°     |

### 配置步骤
1. 配置TIM定时器
2. 设置PWM模式
3. 计算并设置对应角度的占空比
4. 使能PWM输出

## 电机

### 工作原理
- **控制方式**：通过改变电源极性控制旋转方向
- **正转**：电极正接
- **反转**：电极反接

### 配置步骤
1. 配置GPIO为输出模式
2. 使用晶体管或电机驱动芯片
3. 实现正反转控制函数
4. 可配合PWM实现速度控制
