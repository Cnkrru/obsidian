# STM32

---
## STM32F103C8T6外设资源

## 1. 基本信息

| 项目      | 规格                |
| ------- | ----------------- |
| 内核      | ARM Cortex-M3     |
| 主频      | 72MHz             |
| 闪存容量    | 64KB              |
| SRAM容量  | 20KB              |
| 工作电压    | 2.0V - 3.6V（3.3V） |
| 封装      | LQFP48            |
| 引脚数     | 48                |
| GPIO引脚数 | 37                |

---

## 2. GPIO端口

| 端口 | 引脚数 | 引脚号范围 | 特性 |
|------|--------|------------|------|
| PA | 14 | PA0-PA15 (PA15不可用) | 支持5V容忍 |
| PB | 16 | PB0-PB15 | 支持5V容忍 |
| PC | 3 | PC13-PC15 | 支持5V容忍 |
| PD | 2 | PD0-PD1 | 支持5V容忍 |

## 8. 其他外设

| 外设 | 类型 | 特性 |
|------|------|------|
