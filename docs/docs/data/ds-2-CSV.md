# 数据序列化

---
## CSV 基础知识点

---
### 1. 基本结构
- CSV（Comma-Separated Values）是一种纯文本格式，使用逗号分隔值，通常用于存储表格数据。

### 2. 语法规则
- 每行表示一条记录，字段间用逗号分隔。
- 包含逗号、引号或换行符的字段需要用双引号包裹。

### 3. 数据类型
- 所有值默认视为字符串，解析时由应用程序根据需要转换类型。

### 4. 注释
- 标准 CSV 不支持注释，如需注释通常在文件开头添加特殊行（如以 # 开头）。

### 5. 表头
- 第一行通常作为表头，定义各列的名称。

### 6. 分隔符
- 标准分隔符是逗号（,），但也支持其他分隔符（如分号 ;、制表符 \t）。

### 7. 空值
- 连续的逗号表示空值，如 `value,,value` 表示中间字段为空。

### 8. 示例
```csv
name,age,city
John,30,New York
Jane,25,Los Angeles
```
